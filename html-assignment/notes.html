HTML notes

h1
To add headings on the page, you can use the h1 tag for the main heading, going up to h6 for the least important heading.
em
An em tag is used to emphasize a particular word or phrase you want the reader to pay more attention to. The text between the em tags will be in italics
abbr
An abbr tag is used to add abbreviations to certain content. It is required to add a title attribute inside the opening abbr tag to explain what the abbreviation means. The content shown on the page will be abbreviated and underlined with dots. If you pass the mouse over the abbreviation, its meaning will be displayed on the screen.
img
An img tag will add an image to the page. Two attributes are required inside the img tag, which is a singular tag. One is the src attribute, where you will add the path to the image, and the other is the alt attribute, where you add a description of the image. The alt attribute is needed so that if the image doesn't load, the description will be displayed on the screen.
a
An a tag stands for anchors and it is used as links to another page or any other resource as long as it is avaiable on the web. It is required to have an href attribute inside the opening anchor tag to provide the location to the other resource on the web.
